# SoundlapseUACh 

## Sound Time Lapse



### Language
This algorithm was created in Python 3.7.4.

#### Install packages
- Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements to use STL.

```bash
pip install -r requirements.txt  
```
or 
```bash
pip3 install -r requirements.txt  
```
